# Sentinel Ultimate v7.0 🛰️

**Sentinel Ultimate** - это высокоскоростной инструмент сетевой разведки, оптимизированный для Linux. Скрипт эффективно сканирует подсети, определяет живые хосты, сопоставляет их с именами устройств и проверяет открытые критические порты.

## 🚀 Основные возможности
- **Многопоточный движок:** Использование `ThreadPoolExecutor` обеспечивает мгновенное обнаружение устройств.
- **Аудит сервисов:** Проверка стандартных админ-портов: SSH, HTTP, HTTPS, SMB, RDP и других.
- **Обход блокировок Telegram:** Интеллектуальная система доставки отчетов (через прямые запросы, зеркала или личные Cloudflare Workers) для обхода DPI и фильтров провайдера.
- **История сканирования:** Автоматическое сохранение всех результатов в файл `scan_history.log`.
